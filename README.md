# landscaping-business-website
- Based on WordPress Underscores theme, https://rwstone.ca/
- Used VS Code IDE and Laragon as local development environment   
- Underscores watches all SASS files and recompiles them to CSS when they change, refer to https://github.com/automattic/_s
- Toggling navigation menu for smaller screens is handled by mobile-nav-toggle.js
